# 杠杆与清算

了解杠杆交易和清算机制对于成功的交易至关重要。这些概念决定了您的风险敞口、保证金要求和账户健康管理。

## 杠杆交易简介

杠杆允许交易者控制比其资金更大的头寸。例如，使用10倍杠杆，您可以用1,000 USDC控制价值10,000 USDC的头寸。然而，杠杆是一把双刃剑：它可以放大收益，但也会放大损失。

在Triplex Exchange上，交易者可以获得高达25倍的杠杆，让您最大限度地利用市场机会，同时实施适当的风险管理实践。

## 初始与维持保证金

交易保证金有两个关键组成部分：

### 初始保证金

初始保证金是开设和维持杠杆头寸所需的最低抵押品金额。它确保您有足够的资金覆盖潜在损失。

计算公式：
```
初始保证金 = 头寸价值 / 最大可用杠杆
```

例如，一个价值10,000 USDC的头寸，最大杠杆为10倍，将需要1,000 USDC的初始保证金。

初始保证金水平确定了您可以使用的最大杠杆：
- 10倍杠杆 = 10%的初始保证金要求
- 25倍杠杆 = 4%的初始保证金要求

### 维持保证金

维持保证金是账户避免清算必须维持的最低抵押品水平。当账户价值低于这一水平时，清算过程开始。它通常设置为初始保证金要求的一小部分。

计算公式：
```
维持保证金 = 头寸价值 * 维持保证金比率
```

在Triplex Exchange上，维持保证金比率为2.5%，意味着一个价值10,000 USDC的头寸需要至少250 USDC的维持保证金。

## 账户健康与清算

### 账户健康度（保证金比率）

账户健康度是衡量账户离清算还有多远的关键指标。它计算为：

```
账户健康度 = (账户股权 / 维持保证金要求) * 100%
```

其中：
- **账户股权** = 存款余额 + 未实现盈亏(UPNL)
- **维持保证金要求** = 所有开放头寸所需的维持保证金总和

### 账户健康状态指示器：

- **良好（绿色）**：账户健康度 > 142.85%
- **警告（黄色）**：142.85% > 账户健康度 > 111.11%
- **危险（红色）**：111.11% > 账户健康度 > 100%
- **清算（黑色）**：账户健康度 ≤ 100%

<Callout type="error">
当账户健康度降至100%或以下时，您的头寸将被清算以保护系统避免不良债务！
</Callout>

## 清算过程

清算是当交易者的账户价值下降到维持保证金水平以下时的自动平仓过程。以下是清算的工作原理：

1. **触发**：当账户健康度降至100%或以下时触发清算。
2. **平仓**：系统自动关闭账户中的所有头寸。
3. **费用**：收取清算费用，通常为头寸价值的一小部分。
4. **结算**：剩余资金返回到交易者的账户。

### 主动管理风险

为避免清算，可以采取以下策略：

- **降低杠杆**：使用较低的杠杆水平以获得更大的安全边际。
- **增加抵押品**：在市场波动时添加更多抵押品以提高账户健康度。
- **止损订单**：设置止损订单以在价格达到特定水平时自动关闭头寸。
- **监控资金费率**：高资金费率可能会随着时间侵蚀您的保证金，特别是高杠杆头寸。

## 使用风险计算器

Triplex Exchange提供一个风险计算器，让您可以在开仓前评估潜在风险：

- **预计清算价格**：明确价格在哪一点将触发清算。
- **回报潜力**：根据不同价格水平估计潜在收益。
- **风险评估**：评估与杠杆头寸相关的风险。

通过理解这些杠杆和清算机制，您可以更好地管理风险，保护您的资本，并为您的交易策略做出明智的决策。
