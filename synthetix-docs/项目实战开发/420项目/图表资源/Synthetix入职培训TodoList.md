# Synthetix 420 Pool 入职培训计划

## 任务使命 (Mission)
建立对420 Pool与Synthetix V3架构的全面理解，掌握关键交互流程和调用模式，能够独立进行开发和调试工作。

## 背景上下文 (Context)
420 Pool是基于Synthetix V3构建的质押池产品，需要深入理解其架构设计、交互模式和关键功能才能有效开展工作。本培训计划将帮助新成员从零开始，逐步建立对整个系统的全面认识。
