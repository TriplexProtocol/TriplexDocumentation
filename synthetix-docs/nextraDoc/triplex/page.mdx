## FAQ

The Triplex FAQ is a collection of useful questions and answers about the
project. If you have a question that isn't answered here, please
[open a discussion](https://github.com/shuding/triplex/discussions).

<details>
  {/* prettier-ignore */}
  <summary>Can I use Triplex with Next.js `pages` router?</summary>
Triplex 4 only works with the `app` router. Only Triplex 1/2/3 supports the
`pages` router.

</details>

<details>
  <summary>Can I use X with Triplex?</summary>
The answer is "yes" for most things. Since Triplex is just a Next.js plugin, almost all the things
that can be done with React can be done with Triplex. Here are some examples and guides:

- [Use Tailwind CSS](/docs/guide/tailwind-css)
- [Use custom CSS and Sass](/docs/guide/custom-css)
- [Use custom fonts](https://nextjs.org/docs/basic-features/font-optimization)

</details>

<details>
  {/* prettier-ignore */}
  <summary>How can I add a live coding component in Triplex?</summary>
There are libraries like [Sandpack](https://sandpack.codesandbox.io) and
[react-live](https://github.com/FormidableLabs/react-live) that can help you
add live coding components to your MDX.

</details>
