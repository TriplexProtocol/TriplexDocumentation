# TRT 质押常见问题

## 基础问题

### 什么是 TRT 质押？
TRT 质押是将您的 TRT 代币锁定在智能合约中，以获得质押奖励的过程。通过质押，您可以获得协议费用分成、通胀奖励等多种收益。

### 最低质押金额是多少？
最低质押金额为 100 TRT。没有最高限制，但大额质押可以获得额外奖励。

### 质押期限有多长？
- 最短质押期限：7 天
- 推荐质押期限：30 天或更长
- 长期质押可获得额外奖励

## 收益相关

### 质押收益来自哪里？
1. 协议费用分成（约 8-12% 年化）
2. 通胀奖励（约 5-8% 年化）
3. 额外激励（视活动而定）

### 如何领取收益？
- 协议费用：每周自动发放
- 通胀奖励：可随时领取
- 活动奖励：完成指定任务后领取

### 收益率会变化吗？
是的，收益率会根据以下因素动态调整：
- 协议使用率
- 质押总量
- 市场条件
- 治理决议

## 操作指南

### 如何开始质押？
1. 准备足够的 TRT 和 ETH（用于 gas 费）
2. 访问质押页面
3. 选择质押金额和期限
4. 确认交易

### 如何解除质押？
1. 等待锁定期结束
2. 访问质押页面
3. 选择解除质押金额
4. 确认交易

### 质押期间可以转让代币吗？
不可以。质押期间的 TRT 会被锁定在智能合约中，无法转让或交易。

## 安全性问题

### 质押的安全性如何？
- 智能合约经过审计
- 代码开源透明
- 多重签名管理
- 定期安全检查

<Callout type="warning">
虽然我们采取了多重安全措施，但仍建议您了解相关风险，合理分配资产。

### 如何保护我的质押资产？
1. 使用硬件钱包
2. 定期检查账户活动
3. 不分享私钥和助记词
4. 警惕钓鱼网站

## 技术问题

### 为什么我的交易失败了？
常见原因包括：
- ETH 余额不足（gas 费）
- 质押金额超过余额
- 网络拥堵
- 滑点设置过低

### 如何优化 Gas 费用？
1. 选择网络不拥堵时操作
2. 使用 gas 追踪器
3. 批量操作减少交易次数
4. 设置合理的 gas 限制

## 其他问题

### 质押会影响我的投票权吗？
不会。质押的 TRT 仍然可以参与治理投票，且可能获得额外的投票权重。

### 如何参与社区治理？
1. 持有或质押 TRT
2. 访问治理页面
3. 参与提案投票
4. 提交改进建议

### 遇到问题如何寻求帮助？
- 访问官方文档
- 加入社区讨论
- 联系技术支持
- 提交问题反馈
