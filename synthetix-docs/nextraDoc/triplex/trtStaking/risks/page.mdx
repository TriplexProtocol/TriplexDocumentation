# TRT 质押风险说明

<Callout type="warning">
在参与 TRT 质押之前，请仔细阅读并理解以下风险说明。质押加密资产存在多种风险，请在充分了解风险的基础上做出投资决策。
</Callout>

## 市场风险

### 价格波动风险
- TRT 代币价格可能因市场情况发生大幅波动
- 质押期间无法及时响应市场变化
- 可能面临短期或长期的价值损失

### 流动性风险
- 质押期间 TRT 被锁定，无法交易
- 解锁后可能面临流动性不足
- 大额交易可能造成价格影响

## 技术风险

### 智能合约风险
- 智能合约可能存在未知漏洞
- 可能受到黑客攻击
- 技术升级可能带来不确定性

### 网络风险
- 区块链网络可能发生拥堵
- 交易可能被延迟或失败
- Gas 费用可能大幅波动

## 运营风险

### 收益波动风险
- 质押收益率不固定
- 受市场条件影响可能下降
- 通胀奖励会逐年递减

### 政策风险
- 监管政策可能变化
- 可能影响项目运营
- 跨境合规问题

## 用户风险

### 操作风险
- 错误的操作可能导致资产损失
- 私钥丢失将无法找回资产
- 钓鱼网站诈骗风险

### 决策风险
- 质押期限选择不当
- 投资金额分配不合理
- 未充分了解产品机制

## 应急预案

### 发现安全问题
1. 立即停止操作
2. 保存相关证据
3. 联系技术支持
4. 等待官方回应

### 资产异常
1. 检查交易记录
2. 确认钱包安全
3. 报告异常情况
4. 按指引处理
