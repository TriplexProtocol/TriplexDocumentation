# TRT 质押奖励机制

TRT 质押者可以获得多种形式的奖励。本页面将详细介绍各种奖励机制。

## 奖励来源

### 1. 协议费用分成
- 来自交易费用的分成
- 根据质押比例分配
- 每周自动发放

### 2. 通胀奖励
- 协议通胀产生的 TRT 代币
- 按质押时长和数量计算
- 实时累积，可随时领取

### 3. 额外激励
- 特殊活动期间的额外奖励
- 治理投票奖励
- 生态建设奖励

## 奖励计算公式

<Callout type="info">
基础年化收益率 = 协议费用收益 + 通胀奖励率
</Callout>

实际收益计算：
```
实际收益 = 质押金额 × 质押时长 × (基础收益率 + 额外奖励率)
```

## 影响因素

以下因素会影响您的实际收益：

1. 质押金额
- 最低质押要求：100 TRT
- 质押金额越大，获得的奖励比例越高

2. 质押时长
- 最短锁定期：7天
- 较长的质押时间可能获得额外奖励

3. 市场条件
- 协议使用率
- TRT 代币价格波动
- 整体市场状况

<Callout type="warning">
收益率会随市场情况动态调整，不构成任何投资建议。
</Callout>

## 领取奖励

### 自动发放的奖励
- 协议费用奖励每周自动发放到质押者地址

### 需要手动领取的奖励
1. 通胀奖励
2. 活动奖励
3. 治理奖励

<Callout>
建议在奖励累积到一定数量后再领取，以节省 gas 费用。
</Callout>

## 常见问题

### 奖励会自动复投吗？
不会，但您可以手动将领取的奖励再次质押。

### 未领取的奖励会过期吗？
不会，奖励会持续累积直到您领取。

### 如何查看当前收益率？
您可以在质押界面实时查看当前的年化收益率。
