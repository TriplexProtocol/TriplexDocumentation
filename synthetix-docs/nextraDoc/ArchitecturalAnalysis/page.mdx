## 系统概述

Synthetix V3 是一个去中心化的衍生品流动性协议，允许创建合成资产并在链上交易，通过创新的池化流动性机制和灵活的多抵押品系统提供无滑点交易体验。

## 主要业务流程

### 合成资产铸造流程

<Steps>
  1. **账户验证**
     - 用户请求使用账户
     - 账户系统验证权限
  
  2. **抵押品存入**
     - 用户向流动性池存入抵押品
     - 流动性池记录抵押品
  
  3. **价格查询**
     - 系统从 Oracle 获取价格数据
     - Oracle 返回最新价格
  
  4. **抵押率检查**
     - 检查抵押品价值与铸造额度
     - 确认抵押充足性
  
  5. **资产铸造**
     - 创建合成资产
     - 转移资产给用户
     - 更新债务记录
</Steps>

### 交易执行流程

<Steps>
  1. **交易准备**
     - 交易者进行账户授权
     - 验证交易权限
  
  2. **订单提交**
     - 提交交易订单
     - 获取最新价格数据
  
  3. **订单验证**
     - 验证交易条件
     - 执行交易匹配
  
  4. **流动性提供**
     - 请求交易流动性
     - 确认流动性充足
  
  5. **交易完成**
     - 确认交易执行
     - 分配交易费用
</Steps>

### 治理决策流程

<Steps>
  1. **提案阶段**
     - SNX 持有者提交提案
     - 代表理事会初步审议
  
  2. **评估阶段**
     - 理事会讨论和评估
     - 根据提案类型决定流程
  
  3. **投票阶段**
     - 重大变更：提交公投
     - 常规变更：理事会表决
  
  4. **执行阶段**
     - 批准并记录决策
     - 应用参数变更
     - 确认变更完成
</Steps>
